What's inside

Built like a real product, opinionated like a research demo.

Session-based auth, done right

Bcrypt-hashed passwords, server-stored sessions, httpOnly cookies, double-submit CSRF, and per-IP rate limits on sensitive routes.

Hardened uploads

Magic-byte image validation, 8 MB cap, OS-temp file routing — no client filenames touch disk paths.

Cryptographic OTP

Codes via secrets not random, hashed at rest, 10-minute TTL, capped attempts, single-use.

Proximity that respects truth

Custom Haversine, validated bounds, no external dependency. Domain-matched users bypass distance — everyone else doesn't.

Trust-gated discovery

Drop below a trust threshold and visibility revokes itself — no moderator needed for autonomous governance.

Strict security headers

CSP, X-Frame-Options DENY, Referrer-Policy, Permissions-Policy, HSTS in prod, X-Content-Type-Options nosniff.
